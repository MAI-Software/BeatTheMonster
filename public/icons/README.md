# Iconos del juego (placeholders)

Cada icono usa de momento un **emoji** como placeholder. Para sustituirlo por tu
diseño, deja un PNG (fondo transparente, cuadrado) con el nombre exacto:

```
public/icons/
  campaign.png    ⚔️  (Luchar)
  practice.png    🎯  (Práctica)
  tutorial.png    🥊  (Tutorial)
  training.png    💪  (Entrenar)
  equip.png       🧤  (Equipo & Flow)
  gacha.png       🎰  (Gacha)
  challenges.png  📅  (Desafíos)
  ranking.png     🏆  (Ranking)
  coin.png        🪙  (monedas)
  gem.png         💎  (gemas premium)
  flow.png        ⚡  (Estado de Flujo)
  vt.png          ❤️  (VT / vida)
  atk.png         👊  (ataque)
  def.png         🛡️  (defensa)
```

En cuanto exista el PNG, reemplaza al emoji automáticamente. Recomendado: 128×128
o 256×256, PNG con transparencia. Paleta del juego: negro, blanco y rojo.
